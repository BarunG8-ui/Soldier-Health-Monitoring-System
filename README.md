# Soldier-Health-Monitoring-System

## 🔧 Components Used

| Component               | Purpose                                         |
| ----------------------- | ----------------------------------------------- |
| Arduino Uno R3          | Main microcontroller for processing sensor data |
| Pulse Sensor            | Measures soldier's heart rate                   |
| Neo-6M GPS Module       | Tracks real-time geographical location          |
| GSM SIM900A Module      | Sends health and location data via SMS          |
| LDR Sensor              | Detects ambient light conditions                |
| Buzzer                  | Provides emergency alert indication             |
| LEDs                    | Status and warning indications                  |
| Resistors               | Circuit protection and current limiting         |
| 12V Lithium-Ion Battery | Portable power supply for the system            |

## ⚙️ Working Model

1. The Pulse Sensor continuously monitors the soldier's heart rate.
2. The Neo-6M GPS module acquires the current location coordinates.
3. Arduino Uno processes the health and location data.
4. If abnormal health conditions are detected, the system triggers the buzzer and LED alerts.
5. The GSM SIM900A module sends the soldier's health status and GPS coordinates to the control center or rescue team.
6. The rescue team receives the alert and location information through SMS.
7. Using the transmitted coordinates, the injured soldier can be located and rescued quickly.
8. The entire system is powered by a rechargeable 12V Lithium-Ion Battery, making it suitable for field deployment.
